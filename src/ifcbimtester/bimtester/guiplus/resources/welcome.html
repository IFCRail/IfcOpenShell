Welcome

Get started with 3 easy steps

  1. Select your working directory (e.g. IFCRail\IFC-Rail-Unit-Test\0_BIMTester\UT_BIMT_0)
  2. Specify the options
    • the IFC file to validate
    • the Feature file to execute
    • the directory to write Reports to
  3. Click the Run button

Hints

Use arguments -p and -z to have a default working directory and zoom factor on start of the application, e.g.:

startbimtesterplus.exe -p "C:\git\IFCRail\IFC-Rail-Unit-Test\0_BIMTester\UT_BIMT_0" -z 2
